# 功能概览

NavTools 是一个功能丰富的网站导航管理系统，集成了现代化的 UI、强大的管理功能和企业级安全特性。本页面概览所有功能模块。

## 核心功能

### 分组管理

分组是组织网站的基本单元。

- **创建分组**：支持自定义名称、图标（Emoji）
- **编辑分组**：修改名称、图标
- **删除分组**：软删除机制，防止误删
- **公开/私有**：设置分组的可见性（访客模式）

::: tip 公开分组
当分组设置为 "公开" 时，未登录的访客也可以看到该分组及其下的公开站点。
:::

[查看详细文档 →](/features/groups)

### 站点管理

每个站点包含以下信息：

- **标题**：网站名称
- **链接**：URL 地址
- **描述**：简短说明（可选）
- **图标**：支持自动获取（Google Favicon / Icon Horse）或自定义图片 URL
- **分组**：所属分组

::: info 图标获取逻辑
1. 优先使用自定义图标 URL
2. Icon Horse 服务（高质量）
3. Google Favicon 服务（备用）
:::

[查看详细文档 →](/features/sites)

### 拖拽排序

提供所见即所得的排序体验。

1. 点击 "编辑排序" 进入模式
2. 拖动卡片到目标位置
3. 点击 "保存排序" 持久化更改

[查看详细文档 →](/features/sorting)

## 访客模式（v1.1.0）

访客模式允许将部分内容分享给未登录用户。

- **配置开关**：`AUTH_REQUIRED_FOR_READ` (true/false)
- **权限控制**：
  - 分组级别公开/私有设置
  - 站点级别公开/私有设置
- **界面适配**：
  - 访客视图（只读，无管理按钮）
  - 管理员视图（完整功能，可见性标记）

[查看详细文档 →](/features/guest-mode)

## 主题与样式

提供高度可定制的外观设置。

### 暗色模式

- **跟随系统**：自动检测系统偏好
- **手动切换**：通过右上角按钮切换

### 自定义 CSS

管理员可以注入全局 CSS 代码，修改任何界面元素。

```css
/* 示例：修改导航栏背景 */
.MuiAppBar-root {
  background: linear-gradient(to right, #ff00cc, #333399);
}
```

[查看详细文档 →](/features/customization)

## 数据管理

为了防止数据丢失，NavTools 提供了完善的数据导入导出功能。

### 导出数据

- 格式：JSON
- 内容：包含所有分组、站点、排序信息和配置

### 导入数据

- 支持全量导入
- 智能合并（基于名称和 URL）

::: warning
导入前建议先导出当前数据作为备份。导入操作会修改现有数据。
:::

[查看详细文档 →](/features/data-management)

## 认证与安全

- **JWT 认证**：无状态、安全
- **速率限制**：防止暴力破解
- **安全审计**：定期安全检查

[查看详细文档 →](/security/)
